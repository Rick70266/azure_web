我的個人技能樹

1. SQL Server Analysis Services (SSAS)

擅長建置多維度資料模型 (Multidimensional) 與表格式模型 (Tabular)，提供高效能的 OLAP 分析運算，優化大規模數據的查詢反應時間。

2. SQL Server Reporting Services (SSRS)

具備設計、部署及管理企業級分頁報表的能力，能整合多種資料來源，實現自動化報表排程發送與嚴謹的權限控管。

3. Power BI

熟練運用 Power BI 生態系進行資料端到端的視覺化，包含 Dataflow 處理、複合模型建構，以及製作具互動性的高階決策儀表板。

4. DAX 開發

精通 Data Analysis Expressions (DAX) 語言，能撰寫複雜的量值 (Measures) 與計算資料表，處理時間智慧 (Time Intelligence) 與動態篩選上下文分析。

5. Python 開發

運用 Python 進行資料自動化清洗 (Pandas)、統計分析及爬蟲開發，並能將 Python 腳本整合至資料科學工作流程或 Power BI 模型中。

6. Azure SQL Database 建置

具備雲端資料庫管理經驗，能執行雲端資料庫的遷移、安全性設定、彈性集區 (Elastic Pool) 配置及效能調校。
