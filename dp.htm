Microsoft Azure Data & AI (DP 系列)

DP 系列專注於「雲端資料架構」與「人工智慧」，適合具備一定技術底子的工程師。

DP-900: Azure 資料基礎

適合人群：初學者、想了解雲端資料庫的人。
準備方向：理解關聯式 (SQL) 與非關聯式 (NoSQL) 資料在庫 Azure 上的差異。

DP-203: Azure 資料工程師

適合人群：資料工程師 (DE)。
準備方向：深入學習 Azure Synapse, Data Lake, Databricks 與數據安全性管理。

DP-100: Azure 資料科學家

適合人群：資料科學家 (DS)、AI 工程師。
準備方向：學習使用 Azure Machine Learning 訓練、測試與部署機器學習模型。

準備方向建議：

  • 熟練 SQL 語法是 DP 系列的基本功。
  • 熟悉 Azure 雲端入口網站 (Portal) 的操作。
  • 實作大型數據集的遷移與清理專案。
